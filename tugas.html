  • Binding data ke dalam view

Vue.js menggunakan syntax {{ }} seperti terlihat pada gambar diatas, dengan syntax tersebut berarti kita ingin menampilkan sebuah data yang ada dalam Javascript kita ke dalam HTML template.

  • Attribute Binding

Hover your mouse over me for a few seconds to see my dynamically bound title!

Attribute Binding di dalam Vue.js

Seperti disebut sebelumnya, di Vue.js kita menggunakan v-bind untuk melakukan binding ke dalam HTML. Maka jika kita melihat kode diatas berarti kita ingin menambahkan attribute title ke dalam span dengan menggunakan data yang dinamis dari javascript. v-bind pada dasarnya bisa diterapkan pada berbagai attribute di HTML sehingga kita mungkin akan menemui banyak variasi v-bind ini seperti v-bind:src, v-bind:class, v-bind:alt ,v-bind:key, dan lain sebagainya.
Vue.js juga menyediakan shortcut untuk kita mendefinisikan v-bind di dalam HTML yakni dengan menghilangkan bagian v-bind nya, jadi kita bisa menggunakan shortcut seperti :title, :src, :class, :alt dan sebagainya.

  • Two way data binding

Seperti AngularJS, Vue.js juga menyediakan fitur two way data binding yang artinya setiap perubahan pada Javascript akan berpengaruh pada view HTML dan sebaliknya perubahan pada HTML view pun akan mempengaruhi atau mengubah nilai di Javascript nya.
Di Vue.js kita menggunakan v-model untuk melakukan two way binding yang biasanya sering ditempelkan pada sebuah elemen input HTML, seperti gambar berikut :

{{ message }}

Vue.js two-way data binding

  • Event binding

Untuk memanggil suatu event yang telah kita buat di Javascript, Vue.js menggunakan v-on di dalam HTML template diikuti hook event yang akan ditambahkan. Jadi kita bisa menggunakan berbagai variasi v-on seperti v-on:click, v-on:blur, v-on:focus, v-on:keyup dan sebagainya. Vue.js menyediakan juga shortcut untuk melakukan event binding ini yakni menggunakan @, sehingga kita bisa menyingkat seperti : @click, @blur,@focus dan sebagainya.

  1. Computed and Watch property

pada vue terdapat Computed and watch property dimana kita sebuah properti tersebut akan berreaksi jika ada perubahan data.

yang membedakan antara Computed dan watch adalah, pada watch component tersebut memantau satu data secara terus menerus jika terjadi perubahan, sedangkan computed component tersebut memantau semua variabel yang bergantung pada pada komponen tersebut contoh:

{{ fullName }} //watch

Top 3 JavaScript Framework/Library

Angular.js, React.js dan Vue.js adalah teknologi front end yang populer yang saling bersaing satu sama lain. Mereka adalah 3 framework/library javasript terbesar didunia. Berikut adalah beberapa komparasi

 

Angular.js

Angular adalah framework yang di buat dan di maintain oleh komunitas di Google. framework ini menjadi inspirasi dalam pembuata VueJS. Kekurangan dari angular adalah proses pembelajaran framework ini kurang cepat, karena memang framework ini ditujukan kepada proyek proyek besar yang kompleks.

 

React.js

React adalah framework yang di buat dan di maintain oleh komunitas di Facebook. Keuntungan terbesar dari React.js adalah pure-performance, berkat virtual DOM yang secara general tidak beda jauh dengan VUE.

 

Vue.js

Vue adalah pendatang baru dan bertujuan untuk menggabungkan beberapa fitur terbaik dari Angular and React: Ini bertujuan untuk menjadi lebih berperforma dan flexible. Vue mampu belajar dari kesalahan dan keberhasilan Angular and React. Pengadopsian Vue dalam jangka panjang tetap menjadi tanda tanya karena dunia JavaScript tetap sangat fluktuatif.
